* How to use:

Call Jalapeno followed by the category of quote you are looking for. Examples:

  ```ruby
  Jalapeno.movie #generates a random movie quote
  ```

  ```ruby
  Jalapeno.famous #generates a famous historical quote
  ```

  ...and so on
